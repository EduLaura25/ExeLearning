Instrucciones para la entrega

Para entregar la tarea, se debe tener en cuenta lo siguiente:

  1. Nombrar el archivo de tu hoja de cálculo siguiendo este esquema: "NumeroGrupo_Nombre1_Nombre2_Nombre3"
